## Descripción
Proyecto desarrollado en C++ orientado al estudio e implementación de conceptos de programación concurrente.

El trabajo busca aplicar:
- hilos
- sincronización
- Semaforos
- exclusión mutua
- productores y consumidores
- manejo de recursos compartidos
- Control de acceso usando Prioridades
